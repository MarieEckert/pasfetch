# pasfetch
A System fetch program written in Pascal.

## Installation
pasfetch is on the [AUR](https://aur.archlinux.org/packages/pasfetch). You can install it using yay: `yay -S pasfetch` <br>
Alternatively you can clone the AUR package and install it manually using makepkg.

## Building
pasfetch is built using fpc and [mariebuild](https://github.com/MarieEckert/mariebuild). If you do not want to use
mariebuild, you can compile the program using `./build.bash`.

## Usage
See the manpage (`pasfetch.adoc`) / documentation in `doc/`.
